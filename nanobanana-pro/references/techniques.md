## Spatial Anchors & Layout Control

Nano Banana Pro excels with explicit spatial instructions. Use these patterns:

### Positional Anchors

```
Subject: [position relative to frame]
- "centered in frame"
- "positioned in the left third"
- "filling the right side of the composition"
- "in the foreground, slightly off-center"
```

### Depth Layers

```
Foreground: [close objects with detail]
Middle ground: [subject typically here]
Background: [environmental context]

Example:
"In the foreground, a dark luxury perfume bottle with metallic accents.
In the middle ground, she sits on the marble countertop.
In the background, a large mirror reflects the door."
```

### Camera Distance

```
- "extreme close-up on face only"
- "medium close-up, chest and head"
- "medium shot, waist up"
- "full body shot"
- "wide establishing shot showing full environment"
```

### Angle Specifications

```
- "eye-level, direct"
- "slightly low angle, looking up"
- "high angle, looking down"
- "three-quarter view"
- "profile view"
- "over-the-shoulder perspective"
```

## Lighting Mastery

### Natural Light Types

```
Golden Hour:
"warm golden-hour sunlight entering from the right,
soft highlights, gentle falloff, realistic contrast"

Overcast:
"soft diffused overcast light, even illumination,
no harsh shadows, subtle gradients"

Window Light:
"natural window light from the left, side-lit,
high contrast chiaroscuro, warm golden temperature"

Dappled:
"light filtering through leaves/blinds creating
distinct striped shadows across the subject"
```

### Artificial Light Types

```
Flash:
"hard flash photography, bright flash burst visible,
starburst flare, high contrast, hard shadows"

Practical Lights:
"warm practical light from bedside lamp,
mixed with cool moonlight from window"

Ring Light:
"even frontal lighting with catchlights in eyes,
minimal shadows, beauty lighting"
```

### Light Behavior Descriptors

```
Quality: "soft" / "hard" / "diffused" / "harsh"
Direction: "side-lit" / "back-lit" / "frontal" / "top-lit"
Temperature: "warm golden" / "cool blue" / "neutral"
Intensity: "bright" / "dim" / "blown-out highlights"
Falloff: "gentle falloff" / "rapid falloff" / "even"
```

### Shadow Patterns

```
- "venetian blind shadows across legs"
- "gobo pattern shadows on wall"
- "soft undefined shadows"
- "hard-edged shadows"
- "no visible shadows (flat lighting)"
```

## Narrative Elements

### Story Context Patterns

Add a brief narrative to ground the image in reality:

```
"She's been on the balcony for hours.
Bottle was full, now it's not.
The city kept her company.
```

```
- "subtle pout, natural resting"

Details:
```

### Micro-Expression Combinations

```
"Tired but inviting":
eyes: heavy-lidded but still warm
mouth: soft, relaxed, slight upturn
brows: slightly lowered, relaxed
overall: "I'm tired but I'd wake up for you"
```

## Negative Prompt Patterns

### Core Negative Prompts (Always Include)

```
altered face, different identity, idealized beauty,
flawless skin, plastic or waxy texture, CGI,
digital art, illustration, artificial lighting,
broken anatomy, distorted hands, harsh shadows,
overprocessed skin, cartoon style
```

### Anatomy Protection

```
missing fingers, extra fingers, deformed hands,
fused fingers, incorrect finger count,
broken wrists, twisted limbs, extra limbs,
floating limbs, disconnected body parts
```

### Face Protection

```
morphed face, asymmetrical eyes, cross-eyed,
different person, face swap, beauty filter,
smoothed skin, poreless skin, doll-like
```

### Style Protection

```
3D render, CGI, video game, anime, manga,
cartoon, illustration, painting, drawing,
digital art, concept art, unrealistic
```

### Quality Protection

```
low quality, blurry, pixelated, noisy,
artifacts, compression, watermark, text,
logo, signature, border, frame
```
