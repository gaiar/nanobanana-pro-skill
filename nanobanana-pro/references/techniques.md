## Defeating the AI Look

AI-generated images share telltale traits: perfect symmetry, porcelain skin, oversaturated colors, and a synthetic studio sheen. Use these four principles to break through the uncanny valley.

### 1. Break Perfect Symmetry and Composition

AI defaults to centered subjects, balanced framing, and rule-of-thirds precision. Real photos are messier.

```
Composition breakers:
- "subject slightly off-center, tilted frame"
- "awkward crop cutting off part of the head or elbow"
- "background element intruding into the frame edge"
- "slightly crooked horizon, not corrected"
- "one shoulder higher than the other, natural asymmetry"
- "shot from an imperfect angle, as if the photographer shifted mid-click"

Facial asymmetry:
- "one eye slightly narrower than the other"
- "natural asymmetry in jawline and brow height"
- "uneven smile, one corner higher"
- "nose not perfectly centered, slight natural deviation"
```

Avoid: dead-center subjects, perfectly mirrored poses, textbook compositional balance.

### 2. Real Skin, Not Porcelain

AI models gravitate toward flawless, pore-free, uniformly lit skin â€” the "porcelain doll" effect. Push hard for biological texture.

```
Skin texture directives:
- "visible pores on nose and cheeks"
- "natural redness around nostrils and inner eye corners"
- "uneven skin tone, slight blotchiness"
- "fine peach fuzz visible on cheeks in side light"
- "tiny imperfections: a faint scar, a mole, a blemish"
- "dark circles under eyes, natural fatigue"
- "chapped lips with visible texture, not glossy-smooth"
- "sun damage, freckle clusters of varying size and density"
- "visible veins on hands, temples, or inner wrists"
- "skin reacts to environment: goosebumps, flush from cold, sweat sheen"

Explicit exclusions:
- "NOT airbrushed, NOT smoothed, NOT beauty-filtered"
- "no porcelain-doll effect, no waxy texture"
- "no uniform skin tone, no synthetic perfection"
```

### 3. Kill Saturation and Remove AI Sheen

AI images tend to be oversaturated with an artificial glow â€” colors too vivid, highlights too clean, everything coated in a synthetic luminosity.

```
Desaturation and color correction:
- "muted, desaturated color palette"
- "colors slightly faded as if shot on expired film"
- "subdued tones, nothing artificially vivid"
- "natural color response, no HDR boost"
- "lifted blacks, compressed dynamic range"
- "skin tones lean warm and matte, not luminous"

Anti-sheen directives:
- "no artificial glow or bloom on skin"
- "matte finish, no synthetic radiance"
- "dull highlights, not blown-out or pearlescent"
- "light falls off naturally, no global illumination feel"
- "shadows are muddy and imprecise, not clean gradients"

Film stock shortcuts (inherently desaturated):
- "Kodak Portra 400 color response" â€” warm, gentle, organic
- "Fujifilm Superia 400" â€” muted greens, understated
- "VSCO A6 processing" â€” analog, faded, real
- "cross-processed slide film" â€” shifted hues, imperfect
```

### 4. Add Grain and Analog Artifacts

The final layer: physical imperfections that no neural network adds on its own. These break the digital sterility.

```
Film grain:
- "visible film grain, especially in shadows and midtones"
- "medium grain structure consistent with ISO 400-800 film"
- "coarse grain in low-light areas"
- "grain is organic and irregular, not uniform digital noise"

Lens artifacts:
- "subtle chromatic aberration at frame edges"
- "slight lens softness in corners, sharp only at center"
- "natural vignetting, darker edges"
- "lens flare from practical light source, not added in post"
- "minor barrel distortion from wide-angle proximity"

Analog imperfections:
- "faint light leak bleeding warm color from one edge"
- "very slight motion blur on hands or hair, frozen imperfectly"
- "dust speck or hair on the negative, barely visible"
- "scanner texture if applicable"
- "slight color fringing on high-contrast edges"

Camera behavior:
- "shallow depth of field with imperfect focus transition"
- "focus slightly missed â€” sharp on the ear, soft on the eyes"
- "auto-white-balance slightly off, giving a color cast"
- "flash redeye partially corrected, slight residual glow"
```

### Putting It All Together

When applying these principles, layer them â€” don't pick just one:


## Physical Medium Authenticity

AI image models default to clean, polished, digitally perfect output â€” even when asked for "hand-made" or "traditional" media. This section provides the vocabulary and prompt architecture to overcome that default for any physical medium.

### Why Prompts Fail at Authenticity

When you write `"linocut print with visible imperfections"` or `"oil painting with thick brushstrokes"`, the model interprets these as **stylistic flourishes** applied to an otherwise clean image. The result looks like a digital filter â€” clean underneath, with "imperfections" sprinkled on top.

To get genuine authenticity, you must:
1. Make the physical medium the **primary subject** of the prompt (more text, higher position)
2. Describe the **physical process** that creates each imperfection (causal, not decorative)
3. Explicitly **negate the clean default** the model wants to produce

### Prompt Architecture for Physical Media

Structure prompts for physical media in this order:

```
1. MEDIUM AND PROCESS (40% of prompt text weight)
   - What physical medium is this?
   - What tools and materials were used?
   - How does the physical process create imperfections?
   - What does the medium look like when done imperfectly by hand?

2. EXPLICIT NEGATIONS (10% of prompt text weight)
   - What should this NOT look like?
   - "NOT clean, NOT digital, NOT polished, NOT perfect"
   - Negate the specific clean defaults for this medium

3. COMPOSITION AND SUBJECT (30% of prompt text weight)
   - What is depicted in the image?
   - Composition, elements, arrangement

4. COLOR AND MOOD (10% of prompt text weight)
   - Color palette with restrictions
   - Emotional tone

5. NEGATIVE PROMPTS (10% of prompt text weight)
   - Technical exclusions
```

The key insight: **the medium description should outweigh the subject description**. If you write 3 sentences about the UFO and 1 sentence about the print texture, you'll get a clean digital UFO. If you write 1 sentence about the UFO and 3 sentences about ink transfer failures, you'll get an authentic print.

### Causal Language vs. Effect Language

**Effect language** (weak â€” model treats as filter):
```
"visible imperfections"
"hand-printed feel"
"slightly textured"
"with tool marks"
```

**Causal language** (strong â€” model simulates the physical process):
```
"ink fails to transfer in spots because hand-pressing pressure is uneven,
 leaving tiny white dots scattered across dark areas"

"the two color layers are not perfectly aligned because each was pressed
 separately by hand, so pink peeks out slightly offset from black edges"

"edges of the printed area are ragged and torn because the carved block
 is slightly smaller than the paper, leaving irregular ink marks at the margins"
```

The causal version tells the model **why** each imperfection exists, which produces more physically coherent results.

### ALL CAPS Emphasis

Use CAPITALIZED phrases for qualities the model consistently underweights:

```
"THE MOST IMPORTANT ASPECT OF THIS IMAGE IS THAT IT LOOKS LIKE AN
 ACTUAL HAND-PULLED PRINT WITH HEAVY IMPERFECTIONS"

"ONLY faded dark gray and off-white cream â€” no true blacks, no pure whites"

"NOT clean, NOT digital, NOT polished"
```

Reserve caps for 2-3 critical qualities per prompt. Overuse dilutes the effect.
